February 12, 2017: Lazy Evaluation and Observation

There are two parts to the world. The first part is the structure, or scaffolding. The structure is recorded in the tuples connections and contents. Each tuple has a type, seed, and UUID, which collectively can be used to instantiate and initialize or load an object. The challenge is to instantiate these objects only when they are observed.

February 11, 2017: Refactoring

Refactoring like this:

Zone Graph
The world is partitioned into zones.
Scene Graph
Each zone is partitioned in scenes.
Map
Each scene may contain a spatial map.
Agent System
Things that exist in the world and interact with each other.
Dialog System
Agents communicate with one another using dialog.
Commerce System
Agents exchange items using commerce.
Combat System
Agents can fight each other.
Client
Method to view the world using a web browser.
PSMath
A math module.
