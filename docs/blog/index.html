February 11, 2017: Refactoring

Refactoring like this:

Zone Graph
The world is partitioned into zones.
Scene Graph
Each zone is partitioned in scenes.
Map
Each scene may contain a spatial map.
Agent System
Things that exist in the world and interact with each other.
Dialog System
Agents communicate with one another using dialog.
Commerce System
Agents exchange items using commerce.
Combat System
Agents can fight each other.
Client
Method to view the world using an web browser.
Math
A math module.
